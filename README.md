# SYAdSDK目录结构

## Example

运行步骤:

1. 进入Example目录

2. 运行命令: pod install

3. 运行命令: open SYAdSDK.xcworkspace 

## SYAdSDK/Classes

广告SDK获取配置, 判断, 展示, 所有的相关代码. 

## SYAdSDK/SYAdSDK.xcassets

资源图, 用于开屏"摇一摇"动画

# 代码集成

直接将SYAdSDK/Classes, SYAdSDK/SYAdSDK.xcassets两个文件夹以源代码形式集成到项目中
